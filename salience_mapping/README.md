# Content
* **salience_mapping.ipynb**: Notebook containing a function to create salience maps and return a random drawn sample of those.

# Usage Instructions
* Run all cells but the last.
* If you wish to generate a random sample of salience maps allowing side by side comparison of both models and data sets, then call the function: 
    *  **print_salience_sample(testfiles = \<data set filenames>, size = \<Number of samples>)**
